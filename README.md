# SearFood - Producers to Global Consumers

![SearFood Logo](/public/logo.png)

## Overview

**SearFood** is a premium culinary platform dedicated to bridging the gap between authentic Indian producers and global consumers. We specialize in bringing rare, traditional flavors—from the spicy notes of the South to the aromatic blends of the North—directly to your kitchen.

Our mission is to revive and share ancestral wisdom through our curated collection of:

- **Masalas**: Authentic blends like _Gujarati Chai Masala_, _Iyengar Sambar Powder_, and _Everyday Bhaji Masala_.
- **Spices**: Premium single-origin spices including _Lakadong Turmeric_, _Mustard Seeds_, and _True Cinnamon_.
- **Dehydrated Foods**: High-quality, preserved ingredients for global export.

## Features

- **Premium Retail Collection**: Browse our extensive catalog with detailed product descriptions and high-quality imagery.
- **Smart Filtering**: easily filter products by category (Masalas, Spices) or search by name/description.
- **Global Exports**: Information for international bulk orders and logistics.
- **Responsive Design**: A seamless experience across Desktop, Tablet, and Mobile devices.
  - _Mobile Optimization_: Includes a touch-friendly hamburger menu and adaptive layouts.
- **Theme Support**: Default Light Mode for a clean look, with an optional Dark Mode for accessibility.
- **Interactive UI**: Smooth animations, cinematic hero sections, and "Read More" progressive disclosure for long descriptions.

## Technology Stack

Built with modern web technologies for performance and scalability:

- **Framework**: [Vite](https://vitejs.dev/) with [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS with CSS Variables & Flexbox/Grid
- **Routing**: Single Page Application (SPA) logic (Custom History API implementation)
- **Deployment**: Configured for usage with Vercel

## Getting Started

Follow these steps to set up the project locally:

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/Aareevs/SearFood-Website.git
   cd SearFood-Website
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## Folder Structure

- `src/main.ts`: Application entry point and logic (Router, Component Rendering, Theme Management).
- `src/mockData.ts`: Centralized data store for products (Recipes/Spices).
- `style.css`: Global styles, variables, and responsive media queries.
- `public/`: Static assets (Logos, Images).
